# telnet

Учебный проект: реализация telnet-клиента с поддержкой параметров запуска.

## Возможности

Программа поддерживает:
- `-host <string>` - указание хоста для установления соединения, обязательный параметр;
- `-port <N>` - указание порта для подключения, необязательный параметр - по умолчанию выставляется порт "25";
- `-timeout <N>` - таймаут подключения к хосту, можно указать суффиксы как 1s/1m - программа поймет;
- завершение работы клиента как при нажатии "Ctrl+D", так и при закрытии сетевого соединения.
Обработка клиентского ввода данных с клавиатуры и вывод сообщений от сервера реализованы в отдельных горутинах.
